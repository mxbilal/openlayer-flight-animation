Openlayer Integration:
OpenLayers is an open-source JavaScript library for displaying map data in web browsers. 
It provides an API for building rich web-based geographic applications similar to Google Maps and Bing Maps.


Getting Started:
I have created flight animation using openlayer library. It map some static data that is provided to OpenLayers


Prerequisites:
1) Node 
2) Vs code 

Installing:
1) open project in vs code
2) run commmand 'npm install'
3) after installing node-modules run command 'npm start'

Usage:
when map is rendered it shows yellow stroke that start from a point and reach to its destination provided
to it. Click on any point and it will show coordinates.
